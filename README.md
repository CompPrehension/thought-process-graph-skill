# Thought Process Graph AI Agents Skill

This skill helps agents use and analyze Thought Process Graph tools in more depth. It covers CLI usage and source-code analysis for the `its_DomainModel`, `its_Reasoner`, and `its_QuestionGen` projects. If an agent needs documentation, it can fetch it from the [wiki](https://compprehension.github.io).

Prerequisites:
- Install `its_DomainModel` (3.0.0-alpha.7 and newer), `its_Reasoner` (3.0.0-alpha.3 and newer), and `its_QuestionGen` via `mvn clean install`.
